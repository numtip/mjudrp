## Upload Instructions

1. Navigate to the DRP Documents library in the pilot SharePoint site
2. Click **Upload > Files**
3. Select the PDF files for PILOT-001 through PILOT-005
4. After upload, edit metadata for each file using the values from `pilot-metadata-sample.csv`
5. Navigate to the DRP Evidence library
6. Upload EVD-PILOT-001 through EVD-PILOT-005
7. Edit metadata for evidence documents

## File Generation

Pilot documents are not real files — they are placeholders. Create them as minimal PDFs using any PDF generator, or use blank text files with `.pdf` extension for testing purposes.

## Metadata Source

All metadata values are defined in: `pilot/pilot-metadata-sample.csv`
